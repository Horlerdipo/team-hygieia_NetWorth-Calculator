# Team-Hygieia_NetWorth-Calculator
Hygieia Net Worth Calculator is a web application that allows users to calculate their total net worth from their assets and liabilities.

## Built With
* HTML and CSS (Frontend)
* PHP (Backend)
* Figma (UI)
